# LiveSpot

LiveSpot es una plataforma web para descubrir, explorar y gestionar eventos y experiencias en vivo desde una interfaz moderna, clara y responsiva.

## Descripción

La aplicación permite visualizar eventos destacados, navegar por un catálogo, filtrar experiencias por categoría, revisar el detalle de cada evento y acceder a un panel inicial de usuario.

Este proyecto fue desarrollado como una propuesta de producto digital orientada a plataformas de eventos, experiencias en vivo y servicios tipo SaaS.

## Funcionalidades actuales

- Página de inicio con presentación del producto
- Catálogo de eventos
- Búsqueda por nombre o anfitrión
- Filtro por categoría
- Vista de detalle por evento
- Dashboard inicial de usuario
- Navegación entre rutas con React Router
- Interfaz responsiva construida con Tailwind CSS

## Tecnologías utilizadas

- React
- Vite
- React Router
- Tailwind CSS
- JavaScript
